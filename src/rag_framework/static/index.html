    Corrective Policy

    Rewrite Quality Gates

    01

    Intent Preservation

    The rewrite must preserve meaning using embedding similarity, key-term retention, and polarity checks.

    02

    Evidence Improvement

    The rewritten query must improve retrieved evidence after hybrid retrieval, RRF fusion, and reranking.

    03

    Safe Fallback

    If either gate fails, the rewrite is rejected and the system returns to the initial retrieval path.

    04

    Task Planning

    Planner RAG decomposes broad questions, retrieves evidence per sub-question, fuses it, and answers once.
